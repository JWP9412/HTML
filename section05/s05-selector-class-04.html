태그의 Class Selector

태그의 클래스 선택자는 선택된 태그의 클래스로 명시된 태그에 적용된다. 태그 이름 뒤에 마침표(.)를 찍고 클래스 이름으로 정의한다. 태그의 속성에 class = "클래스이름"을 명시하면 클래스 선택자가 적용된다.

Ex) h4.redtext{someThing;}

예제는 모든 hn 태그가 클래스명으로 redtext를 지정되어 있지만 태그(h4)만 스타일 이 적용된다.
