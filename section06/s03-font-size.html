폰트 크기(font-size)

폰트 크기는 절대크기, 상대크기, 백분율, 크기 지정방식이 있다.

  • 절대크기 : 브라우저에서 지정한 글자 크기 (xx-small,x-small,small,medium,x-large,xx-larger)
  • 상대크기 : 부모의 글자 크기를 기준으로 더 크게 표시하거나 더 작게 표현한다. (larger, smaller)
  • 백분율 : 부모의 글자 크기를 기준으로 해다하는 %를 계산해서 표시한다.
  • 크기 : 브라우저와 상관없이 글자 크기를 직접지정
  • em: 해당하는 글꼴의 대문자 M의 너비를 기준으로 크기를 조절
  • ex: 해당하는 글꼴의 소문자 x의 높이를 기본으로 크기를 조절(x-height)
  • px: 모니터에 따라서 상대적 크기를 조절(픽셀)
  • pt: 일반 문서에서 많이 사용하는 단위(포인트)
