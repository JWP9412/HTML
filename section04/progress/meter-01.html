Meter 태그

Meter 태그는 값이 차지하는 크기를 표시한다.
전체 크기 중에서 얼마나 차지하는지 표현할 때 사용한다.
예) 하드 디스크 전체 용량중에서 사용량

  • value : 범위에서 차지하는 값
  • min : 범위에서 최솟값, 디폴트 0
  • max : 범위에서 최댓값, 디폴트 1
  • low : 범위에서 낮은값, 값이 작다는 의미
  • high : 범위에서 놎은값, 값이 높다는 의미
  • optimum : 범위에서 적정값,
    • optimum 값이 high 값보다 크면 value 값이 클수록 좋음
    • optimum 값이 low 값보다 작다면 value 값이 작을수록 좋음
