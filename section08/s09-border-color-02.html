Border-color : 색상

  • 두께 : thin < medium < thick
  • 방향 : top -> right -> bottom -> left
  • none : 기본값, 테두리 없음
  • hidden : 테두리가 나타나지 않음, border-collapse:collapse 일 경우 다른 테두리도 표시
  • dashed : 짧은 점선
  • dotted : 점선
  • double : 이중선
  • groove : 홈이 파인 창 테두리
  • inset : 창 테두리 형태
  • ridge : 테 테두리 형태
  • solid : 실선
