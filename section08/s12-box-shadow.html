Box shadow

박스에 그림자를 준다

box-shadow : none | 그림자값 [,그림자값];

그림자값 : 수평거리 수직거리 흐림정도 번짐정도 색상 inset

  • 수평거리 : 그림자의 수평 옵셋 거리, 양수(오른쪽) 음수(왼쪽)
  • 수직거리 : 그림자의 수직 옵셋 거리, 양수(오른쪽) 음수(왼쪽)
  • 흐림정도 : 그림자의 흐림정도(blur radius)
  • 번짐정도 : 그림자의 번지는 정도
  • 색상
  • inset : 안쪽 그림자
